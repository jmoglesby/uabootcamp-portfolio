Temperature Conversion

This Ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit.

def convert_celsius_to_fahrenheit cels
    cels * 1.8 + 32
end

Foobar

This Ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern.

puts "How many turns of FooBar should we play?"
turns = gets.chomp.to_i

def foobar turns
    current_turn = 1
    output = []
    
    turns.times do
        phrase = ""
        if current_turn % 3 == 0
            phrase << "Foo"
        end

        if current_turn % 5 == 0
            phrase << "Bar"
        end

        if phrase != ""
            output << phrase
        end

        if current_turn % 3 != 0 && current_turn % 5 != 0
            output << current_turn
        end

        current_turn += 1
    end
    return output
end

puts foobar(turns).inspect
